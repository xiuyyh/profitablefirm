# **App Name**: PROFITABLEFIRM

## Core Features:

- Secure Investor Dashboard: Provide a secure login for investors to access a personalized overview of their portfolio.
- Investment Input & Management: Enable investors to easily add, edit, and categorize various investment types (stocks, bonds, funds) within their portfolio.
- Real-time Performance Metrics: Display up-to-date values, daily gains/losses, and overall returns for individual investments and the entire portfolio.
- Historical Performance Visualization: Present portfolio performance over various timeframes using clear, interactive charts and graphs.

## Style Guidelines:

- Primary color: A deep, professional blue (#1F5799) symbolizing trust and stability, serving as a focal point for interactive elements and key information.
- Background color: A subtle, desaturated light blue (#EBF0F4) creating a clean and reassuring foundation that enhances readability without distraction.
- Accent color: A vibrant aqua (#27C5E0) providing clear contrast for calls to action, highlights, and status indicators, injecting a dynamic feel.
- Headline and Body Font: 'Inter' (sans-serif) for its modern, objective, and highly legible characteristics, ideal for displaying financial data and detailed information across the application.
- Employ a minimalist set of outline-style icons that clearly represent financial concepts (e.g., charts, dollar signs, arrows) for intuitive navigation and data representation.
- Adopt a clean, grid-based layout with generous white space to present complex financial data in an organized and easily digestible manner, ensuring responsive adaptation across devices.
- Implement subtle, fluid animations for data loading, chart transitions, and navigation interactions to enhance user engagement and provide smooth feedback without being distracting.
